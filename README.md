# 進級制作

進級制作では三重県伊勢市にある**日昇工業所**のサイトを制作しました。

## 使用技術
HTML / CSS（Sass） / JavaScript / microCMS

## 使用ツール
Visual Studio Code / Figma / Photoshop / Illustrator / Formspree / ChatGPT

## デザイン
### クライアントの要望
- 下からふわっと表示される演出を希望されていたため、フェードインアニメーションを実装しました。
- 青と白を基調とし、アクセントとして黄色を取り入れました。
- 堅苦しくなりすぎず、信頼感が伝わるデザインを求められました。

## 想定ユーザー
- 日昇工業所の取引先企業や協力会社

## 制作目的
配管工事を行う協力会社としての信頼感を伝えるため、コーポレートサイトを制作しました。

## こだわった点
- Tabキーによるキーボード操作に配慮し、フォーカス表示を実装しました。
- 静的サイトでもmicroCMSを導入し、管理者がお知らせを更新できるようにしました。
- 静的サイトではサーバー処理が行えないため、処理を代行する Formspree を導入し、お問い合わせフォームを実装しました。
- 表示速度や使用感の向上を目的に、PageSpeed Insights のスコアを意識して制作しました。

## microCMS
- 管理者画面では API スキーマを設計し、不要な項目を入力できないようにしました。
- お知らせは最新3件のみ表示されるようにし、情報が過多にならないよう配慮しました。
